# TTS Playground

A practical repository for experimenting with open-source Text-to-Speech models, inference setup, Gradio-based testing, latency measurement, and sequential vs batched generation benchmarks.

This repo documents model-specific setup steps, required dependencies, execution commands, troubleshooting notes, and benchmark results so that different TTS systems can be tested and compared in a reproducible way.
